# 🎭 Gogol MAS: Multi-Agent System для реставрации второго тома «Мёртвых душ»

**Версия:** 0.1.0  
**Язык:** Python 3.10+  
**Архитектура:** Трёхуровневая (RAG + LLM + FSM)

---

## 📖 Описание

Проект по реставрации второго тома «Мёртвых душ» Гоголя с использованием многоагентной системы (MAS) на основе fine-tuned языковой модели.

### Трёхуровневая архитектура

```
┌────────────────────────────────────────────────────┐
│  УРОВЕНЬ 1: RAG (Декорации)                        │
│  • Исторические документы                          │
│  • Черновики Гоголя                                │
└────────────────────┬───────────────────────────────┘
                     │
┌────────────────────▼───────────────────────────────┐
│  УРОВЕНЬ 2: LLM (Душа стиля)                       │
│  • Fine-tuned rugpt3small (перплексия 42.73)       │
└────────────────────┬───────────────────────────────┘
                     │
┌────────────────────▼───────────────────────────────┐
│  УРОВЕНЬ 3: FSM (Поведение)                        │
│  • Локальные FSM персонажей                        │
│  • Глобальный FSM (сюжет)                          │
└────────────────────────────────────────────────────┘
```

## 🎯 Агенты

### Чичиков
- **Психотип:** Обходительный, хитрый, целеустремлённый авантюрист
- **Цель:** Установить контакт и намекнуть на выгодную сделку
- **Стратегии:** льстить, умасливать, настаивать, предложить выгоду

### Тентетников
- **Психотип:** Меланхоличный, бездеятельный, рефлексирующий помещик
- **Цель:** Преодолеть меланхолию и найти смысл в общении
- **Стратегии:** наблюдать, сомневаться, отстраняться, согласиться

## 📊 Технические детали

| Компонент | Технология |
|-----------|------------|
| **Модель** | ai-forever/rugpt3small_based_on_gpt2 |
| **Fine-tuning** | QLoRA (4-bit) |
| **RAG** | ChromaDB + sentence-transformers |
| **Векторная база** | data/vector_db/ (10 чанков) |
| **Перплексия** | 42.73 (5 эпох обучения) |
