# Визуализация проекта DeadSouls

## Обзор проекта
Проект **DeadSouls** — это система тонкой настройки языковой модели для генерации текста в стиле Николая Гоголя. Проект использует современные методы дообучения (fine-tuning) с применением LoRA (Low-Rank Adaptation) для адаптации русскоязычной модели к литературному стилю XIX века.

## Архитектура проекта

### Структура каталогов
```

├── generate.py                 # Генерация текста
├── evaluate_model.py           # Оценка модели
```

## Таблица 1: Конфигурация обучения

| Параметр | Значение | Описание |
|----------|----------|----------|

| Эпохи | 3 | Планируемое количество эпох |

## Ключевые выводы

1. **Проект успешно настроен** для обучения на CPU с ограниченными ресурсами.
2. **Датасет достаточного размера** (1,664 абзаца) для тонкой настройки модели 125M параметров.
3. **Первая эпоха показала значительное улучшение** (снижение loss на 17.8%).
4. **Генерация текста работает**, но требует дальнейшего обучения для улучшения качества.
5. **Архитектура гибкая** — позволяет переключаться между разными моделями и методами адаптации.

## Рекомендации по улучшению

1. **Увеличить количество эпох** до 7 для лучшего усвоения стиля.
2. **Добавить разнообразие в датасет** — включить больше произведений русской классики XIX века.
3. **Оптимизировать гиперпараметры** — подобрать оптимальные значения learning rate, batch size.
4. **Внедрить early stopping** для предотвращения переобучения.
5. **Добавить метрики оценки** — BLEU, ROUGE для количественной оценки качества генерации.
