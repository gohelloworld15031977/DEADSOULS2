## 🚀 Как использовать

### Режим 1: Интерактивный чат
```bash
python rag_gogol.py
```

**Пример:**
```
Вы: Какие законы о крепостном праве были при Александре I?
Гоголь: [Поиск в исторической базе]
      Указ о вольных хлебопашцах (1803)...
```

### Режим 2: Тестирование
```bash
python rag_gogol.py --test
```

### Режим 3: Программное использование
```python
from rag_gogol import GogolRAG

rag = GogolRAG()

# Генерация с RAG
answer, source = rag.generate_with_rag("Ваш вопрос")
print(f"Источник: {source}")
print(f"Ответ: {answer}")

# Поиск только в базе
context = rag.retrieve_from_db("законы о крепостных")
print(context)

# Поиск в интернете
from search_module import InternetSearch
search = InternetSearch()
result = search.search("Гоголь Мёртвые души")
```
